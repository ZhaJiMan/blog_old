猴子也能懂的频谱分析入门

2019-11-08

引言

高等数学里介绍过傅里叶变换，能把时域上的函数变换到频域上，即得到时间信号的频谱，从而确定不同频率的信号对总的时间信号的贡献有多大。但是实际上我们获得的数据都是离散的采样点，所以要利用离散傅里叶变换来处理离散时间信号。

一般的数字信号处理教材都会详细介绍离散傅里叶变换的知识，然而，这些教材术语用得天花乱坠，概念和推导层出不穷，但就是不愿意多讲一句离散傅里叶变换结果的意义和应用。举个例子：对实信号做离散傅里叶变换得到的复系数只有前一半是有用的，因此画图时可以只画前一半，而千万不要把后半段对称的图像当作高频的频谱——单就这样一个简单的事实，就没有哪本教材愿意提一嘴。这些教材实在是超过了笔者猴子般的理解能力。

因此，这里总结了学习过程中遇到的问题，用自己的语言重新复述了一遍频谱分析最最基本的入门知识，以方便日后回顾。

本文将会依次介绍：

  • 实离散傅里叶变换
  • 离散傅里叶变换
  • 离散傅里叶变换与傅里叶变换之间的关系
  • 功率谱
  • Python 中的离散傅里叶变换

正文内容主要参考了大崎顺彦的《地震动的谱分析入门》，省略了大量枯燥的推导过程，感兴趣的读者可以去查阅那本书或者相关教材。

实离散傅里叶变换

离散傅里叶变换

为了获得更为简洁的变换式子，接下来把实离散傅里叶变换改写成离散傅里叶变换。这里只简述推导思路：先用复数的欧拉公式把正弦函数和余弦函数改写成复指数，通过关系式

离散傅里叶变换与傅里叶变换之间的关系

上面得出的离散傅里叶变换在时域和频域上都是有限个离散点，而理论推导中常用的傅里叶变换在时域和频域上都是连续的，如何把这两种变换的结果联系在一起呢？这一节便要指出，离散傅里叶变换得到的系数乘上采样持续时间，是对连续频谱的采样的估计。下面是推导过程。

首先引入信号能量的概念。假设 x(t)x(t)x(t) 是通过 1 Ω1\,\Omega1Ω 电阻的电流，那么根据电热公式，全时间的能量为

∫−∞∞x2(t)dt=∫−∞∞∣X(f)∣2df\int_{-\infty}^{\infty} x^2(t) dt = \int_{-\infty}^{\infty} |X(f)|^2 df ∫−∞∞​x2(t)dt=∫−∞∞​∣X(f)∣2df

用到的 Python 代码如下

import numpy as np
import matplotlib.pyplot as plt

# 采样信息
N = 256
fs = 50
dt = 1/fs
T = N*dt

# 取信号采样
a = 10
t = np.arange(N)*dt
x = np.sinc(a*(t-T/2))

# 设置频率
f = np.fft.fftfreq(N, dt)
f = np.fft.fftshift(f)

# 计算FFT
Ck = np.fft.fft(x)
Ck = np.fft.fftshift(Ck)
Fk = dt*Ck

# 计算理想频谱
def Xf(f, a):
    res = np.zeros(f.size)
    indx = (f>=-a/2) & (f<=a/2)
    res[indx] = 1/a
    res[~indx] = 0

    return res

# 画出信号图像
plt.figure(figsize=(18, 5))
plt.subplot(121)
plt.plot(t, x)
plt.xlabel('t [$s$]', size='large')
plt.ylabel('x(t)', size='large')

# 画出频谱图
plt.subplot(122)
plt.plot(f, np.abs(Fk))
plt.plot(f, Xf(f, a), 'r--', alpha=0.8)
plt.xlabel('frequency [$Hz$]', size='large')
plt.ylabel('$X(f)$', size='large')

plt.show()

# 计算数值积分
S = np.sum(np.abs(Fk))/T
print(np.abs(S-1)*100)
