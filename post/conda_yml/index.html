导出和导入 Conda 环境，以及常用配置

2019-09-12

首先conda activate <envname>，进入相应的环境。然后导出环境

conda env export > environment.yml

生成 yml 配置文件。以后导入时，直接在 base 环境下执行

conda env create -f environment.yml

即可。如果有冲突，可以尝试删去相应库的版本号。
